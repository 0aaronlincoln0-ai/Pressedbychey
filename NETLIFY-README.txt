Pressed by Chey - Netlify Upload

Fastest deploy:
1. Go to https://app.netlify.com/drop
2. Drag the whole folder named UPLOAD-THIS-TO-NETLIFY-ROOT-IMAGES-FIX into the upload box, or upload the matching zip file.
3. Do not upload only index.html. Upload the full folder/zip with all image files.
4. After Netlify publishes, open /asset-test.html on the Netlify link. If the test images show there, the website images will show too.

Important:
- This is a static website, so it can be uploaded directly.
- Customer login, guest checkout, saved sizes, saved products, and order history currently use browser storage for this demo version.
- Admin login is through the Account sign-in: admin / chey2026. The Admin tab appears after admin sign-in.
- The admin page includes a Design Studio tab for private nail ideas, maker notes, color recipes, materials, inspiration photos, and a one-nail sketch pad with realistic press-on nail shapes, polish, aura, glitter, 3D charms, stay-in-line drawing, and AI-style assist tools.
- Color & Style photo uploads use smart nail extraction so the product photo becomes a cleaner nail texture on the hand preview.
- Admin photo/text edits are saved in that browser only. To make those edits live for everyone, export the edits and rebuild/redeploy the site, or connect a backend/dashboard.
- For real public customer accounts and payments, connect the site to a secure backend/database and checkout provider.
